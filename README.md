## Поступила просьба ответить на 9 главных вопросов, а также проанализировать продажи квартир на вторичном рынке в городе Хабаровск. Вопросы:

1. Заметно ли увеличение кол-ва квартир в более новых домах?   
2. Какая вилка для большинства квартир в городе?   
3. Какая максимальная / минимальная/ средняя /медианная цена за квартиру любого класса?
4. Какая максимальная / средняя / минимальная площадь квартиры?
5. Какая максимальная / минимальная / средняя цена за кв. метр?
6. Какого сегмента квартир больше низкого/среднего/высокого?
7. Если зависимость цены квартиры от площади по сегментам?
8. Какой самый популярный ремонт в квартирах по сегментам?
9. Есть ли тенденция увеличения кол-ва объявлений с 2020 по 2025 в связи с введением программы «Дальневосточная ипотека»?

## Информация об исходных данных: 

1. Цена - (тип данных: integer)
2. Цена за квадратный метр - (тип данных: float)
3. Адрес - (тип данных: string)
4. Район - (тип данных: string)
5. Комнаты - количество комнат (тип данных: float)
6. Площадь - (тип данных: float)
7. Жилая площадь - (тип данных: float)
8. Кухня - площадь кухни (тип данных: float)
9. Этаж - (тип данных: integer)
10. Тип сделки - условия сделки (тип данных: string)
11. Тип жилья - квартира или иное жилье (тип данных: string)
12. Ремонт - наличие, вид, отсуствие (тип данных: string)
13. Год постройки - год постройки здания (тип данных: integer)
14. Количество этажей - количество этажей в здании (тип данных: integer)
15. Ссылка на объявление - (тип данных: string)
16. Координаты - список из долготы и широты (тип данных: string)

## Запуск:   

1. Клонирование репозитория   
``` git clone link ```
2. Переход в директорию   
``` cd src ```
3. Активация виртуального окружения     
``` .\venv\Scripts\Activate.ps1 ```
4. Установка зависимостей   
``` pip install -r requirements.txt ```
5. Запуск скрипта парсинга    
``` python parsing_data.py ```
6. Переход в Яндекс.Кабинет для получения API     
``` https://developer.tech.yandex.ru/services ```    
6.1. Выбор JavaScript API и HTTP Геокодер    
7. Создание yaml файла для хранение API    
8. Запуск скрипта для парсинга геоточек    
``` python parsing_geopoints.py ```
9. Переход в родительскую папку
``` cd .. ```
10. Запуск скрипта для объединения двух табличек: таблицу с геоточками и основную таблицу   
``` python main.py ```
